Dependencies¶

These are the libraries that the GUI app currently uses in conjunction with the python programming language.

  • TensorFlow allows the use and training of the machine learning model on the background.

  • pydicom allows the application to read the DICOM files and extract the metadata from the header.

  • pandas organizes the metadata into a neat format for plotting.

  • matplotlib allows for plotting the data in an easy to observe format.

  • Tkinter is the base GUI library that was used to make the application.
